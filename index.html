Mathematical Induction

Topic: Proofs & Logical Deduction

The Logic of Induction

We use mathematical induction generally when there are infinite cases and therefore we cannot test every single piece of evidence. Think of it like climbing an endless staircase. If you can step onto the first rung, and if being on any one rung guarantees you can reach the next, you can climb as high as you want.

The 3-Step Algorithm

  1. Base Case: Prove \(P(\text{initial})\) is true. (Stepping onto the first rung).
  2. Assumption: Assume \(P(k)\) is true. (Standing on the \(k\)-th rung).
  3. Inductive Step: Prove \(P(k+1)\) is true.
    *This step absolutely requires a mathematical substitution from the assumption!*

Mathematical Induction is based on the reasoning that if the first case is true (Step 1) and every next case is triggered by the previous one (Steps 2 → 3), then the statement is always true. It is the mathematical equivalent of the domino effect.

Example 1 Mark Scheme

Let \(P(n)\) be the proposition that \(\displaystyle\sum_{i=1}^n (2i-1) = n^2\).

  • Base Case:

    For \(n=1\): LHS = \(2(1)-1 = 1\). RHS = \(1^2 = 1\).

    Since \(1=1\), \(P(1)\) is true. (A1)

  • Assumption:

    Assume \(P(k)\) is true for some \(k \in \mathbb{Z}^+\).

    \(\displaystyle\sum_{i=1}^k (2i-1) = k^2\) (M1)

  • Inductive Step:

    Prove \(P(k+1)\): \(\displaystyle\sum_{i=1}^{k+1} (2i-1) = \displaystyle\sum_{i=1}^k (2i-1) + (2(k+1)-1)\) (M1)

    Substitute assumption: \(= k^2 + 2k + 2 - 1\) (M1)

    \(= k^2 + 2k + 1\) (A1)

    \(= (k+1)^2\) (A1)

  • Conclusion:

    Since \(P(1)\) is true, and \(P(k) \implies P(k+1)\), then by the principle of mathematical induction, \(P(n)\) is true for all \(n \in \mathbb{Z}^+\). (R1)

Example 2 Mark Scheme

Let \(P(n)\) be the proposition that \(\displaystyle\sum_{i=1}^n \frac{1}{i(i+1)} = \frac{n}{n+1}\).

  • Base Case:

    LHS: \(\frac{1}{1(2)} = \frac{1}{2}\). RHS: \(\frac{1}{1+1} = \frac{1}{2}\). \(P(1)\) is true. (A1)

  • Assumption:

    Assume \(P(k)\) is true. \(\displaystyle\sum_{i=1}^k \frac{1}{i(i+1)} = \frac{k}{k+1}\). (M1)

  • Inductive Step:

    Add \((k+1)\)-th term to both sides: \(\frac{k}{k+1} + \frac{1}{(k+1)(k+2)}\) (M1)

    Common denominator: \(\frac{k(k+2) + 1}{(k+1)(k+2)}\) (M1)

    Expand numerator: \(\frac{k^2+2k+1}{(k+1)(k+2)}\) (A1)

    Factor numerator: \(\frac{(k+1)^2}{(k+1)(k+2)}\) (A1)

    Simplify: \(\frac{k+1}{k+2} = \frac{k+1}{(k+1)+1}\). (A1)

  • Conclusion:

    Standard induction concluding statement. (R1)

Example 3 Mark Scheme

Let \(P(n)\) be the proposition that \(5^n - 1 = 4A\) for some \(A \in \mathbb{Z}^+\).

  • Base Case:

    \(5^1 - 1 = 4\). \(4 = 4(1)\), which is divisible by 4. \(P(1)\) is true. (A1)

  • Assumption:

    Assume \(P(k)\) is true. \(5^k - 1 = 4A \implies 5^k = 4A + 1\). (M1)

  • Inductive Step:

    Consider \(5^{k+1} - 1 = 5 \cdot 5^k - 1\) (M1)

    Substitute assumption: \(5(4A + 1) - 1\) (M1)

    \(= 20A + 5 - 1 = 20A + 4\) (A1)

    \(= 4(5A + 1)\). Since \((5A+1) \in \mathbb{Z}^+\), it is divisible by 4. (A1)

  • Conclusion:

    Standard induction concluding statement. (R1)

Example 4 Mark Scheme

Let \(P(n)\) be \(6^n + 4 = 5A\).

  • Base Case: \(6^1 + 4 = 10 = 5(2)\). True. (A1)
  • Assume: \(6^k + 4 = 5A \implies 6^k = 5A - 4\). (M1)
  • Inductive: \(6^{k+1} + 4 = 6 \cdot 6^k + 4\) (M1)
  • \(= 6(5A - 4) + 4\) (M1)
  • \(= 30A - 24 + 4 = 30A - 20\) (A1)
  • \(= 5(6A - 4)\). Divisible by 5. (A1) + (R1) for conclusion.

Example 5 Mark Scheme

Let \(P(n)\) be \(8^n - 3^n = 5A\).

  • Base Case: \(8^1 - 3^1 = 5\). True. (A1)
  • Assume: \(8^k - 3^k = 5A \implies 8^k = 5A + 3^k\). (M1)
  • Inductive: \(8^{k+1} - 3^{k+1} = 8 \cdot 8^k - 3 \cdot 3^k\) (M1)
  • \(= 8(5A + 3^k) - 3 \cdot 3^k\) (M1)
  • \(= 40A + 8 \cdot 3^k - 3 \cdot 3^k = 40A + 5 \cdot 3^k\) (A1)
  • \(= 5(8A + 3^{k-1})\). Divisible by 5. (A1) + (R1) for conclusion.

Example 6 Mark Scheme

Let \(n = 2m\) where \(m \in \mathbb{Z}^+\). Proposition: \(2m(2m+2) = 4A\).

  • Base Case: For \(n=2\), \(2(2+2) = 8 = 4(2)\). True. (A1)
  • Assume: For even \(k\), \(k(k+2) = 4A\). (M1)
  • Inductive: Next even number is \(k+2\). Consider \((k+2)((k+2)+2) = (k+2)(k+4)\) (M1)
  • Expand: \(k^2 + 6k + 8 = (k^2 + 2k) + 4k + 8\) (M1)
  • Substitute: \(k(k+2) = 4A \implies 4A + 4k + 8\) (A1)
  • \(= 4(A + k + 2)\). Divisible by 4. (A1) + (R1) for conclusion.

Example 7 Mark Scheme

Let \(P(n)\) be \(4^{n+1} + 5^{2n-1} = 21A\).

  • Base Case: \(4^2 + 5^1 = 16 + 5 = 21\). True. (A1)
  • Assume: \(4^{k+1} + 5^{2k-1} = 21A \implies 4^{k+1} = 21A - 5^{2k-1}\). (M1)
  • Inductive: \(4^{(k+1)+1} + 5^{2(k+1)-1} = 4^{k+2} + 5^{2k+1}\) (M1)
  • Rewrite: \(4 \cdot 4^{k+1} + 5^2 \cdot 5^{2k-1}\) (M1)
  • Substitute: \(4(21A - 5^{2k-1}) + 25 \cdot 5^{2k-1}\) (A1)
  • \(= 84A - 4 \cdot 5^{2k-1} + 25 \cdot 5^{2k-1} = 84A + 21 \cdot 5^{2k-1}\)
  • \(= 21(4A + 5^{2k-1})\). Divisible by 21. (A1) + (R1) for conclusion.
