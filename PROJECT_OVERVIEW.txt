1. WHAT IS CLEVERPLATFORM?
────────────────────────────────────────────────────────────────────────────────
CleverPlatform is a web-based educational platform for IB Diploma Programme
mathematics courses (AAHL & AIHL). It is designed for use in class and at home
by students, teachers, and parents.

It will replace scattered tools with a single platform that includes:
  • Interactive digital textbook lessons
  • Past-paper question bank (synced from the MSA Grader project)
  • Assignments, submissions, and AI-assisted grading
  • Exam builder
  • Gradebook and student progress tracking
  • Parent access portal

4. DATABASE SCHEMA
────────────────────────────────────────────────────────────────────────────────
The database schema unifies tables from the MSA Grader project with new
CleverPlatform tables. All tables are defined in:
  • SQL:  supabase/migrations/001_initial_schema.sql
  • Types: src/types/database.ts

  [MSA] Existing tables (from MSA_Grader-API_Backend):
  ────────────────────────────────────────────────────
    questions          IB past-paper questions (code, year, paper, level, marks)
    students           Student names and emails
    exams              Exam metadata (code, date, duration, class)
    exam_questions     Which questions are on which exam
    grades             Teacher/AI marks per student per question per exam
    student_responses  Student self-reported marks
    box_coordinates    Coordinates for exam question cropping

  [NEW] CleverPlatform tables:
  ────────────────────────────────────────────────────
    profiles           User profiles with roles (teacher/student/parent/admin)
    courses            Courses (AAHL, AIHL)
    enrollments        Student-course enrollment
    topics             Topics within a course
    lessons            Lessons within a topic
    assignments        Teacher-created assignments
    submissions        Student submissions with AI + teacher grades
    parent_links       Parent-student linking via registration codes

  Row-Level Security (RLS) is enabled on ALL tables with policies for:
    • Students can view their own data
    • Teachers/admins can view and manage all data
    • Parents can view their linked student's data

  Seed data included:
    • Two courses: IBDP AAHL and IBDP AIHL
    • Admin profile for clevermathematics@gmail.com


5. AUTHENTICATION
────────────────────────────────────────────────────────────────────────────────

8. HOW THE MSA GRADER CONNECTS
────────────────────────────────────────────────────────────────────────────────
  The MSA_Grader-API_Backend is a Google Apps Script project that:
    • Reads IB past-paper questions from Google Sheets
    • Uses Mathpix OCR to read student handwritten work
    • Grades student work using a 5-strategy AI engine (SRG_Grader.js)
    • Syncs all data to Supabase via SupabaseSync.js

  CleverPlatform shares the SAME Supabase database instance:
    • The MSA Grader writes: questions, exams, grades, student_responses
    • CleverPlatform reads those tables AND adds its own tables
    • Both projects use the same Supabase URL and anon key
